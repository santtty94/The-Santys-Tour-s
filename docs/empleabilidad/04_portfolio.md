## Proyecto principal · The Santy's Tours

**Repositorio:** [github.com/santtty94/The-Santys-Tour-s](https://github.com/santtty94/The-Santys-Tour-s)

### Qué es
Portal web y infraestructura digital completa para una agencia de actividades turísticas en Barcelona. El proyecto integra todos los módulos del ciclo en una solución tecnológica cohesionada orientada a un negocio real.

### Qué incluye
- **Infraestructura de red** diseñada con Cisco Packet Tracer: topología, direccionamiento IP, VLANs
- **Arquitectura cloud en AWS**: servicios desplegados, gestión DNS, balanceadores de carga
- **Base de datos relacional**: modelo E/R, scripts SQL, gestión de reservas y clientes, política de backups
- **Portal web**: estructura HTML/CSS, documentación XML
- **Análisis de hardware**: propuesta de equipamiento para la infraestructura física
- **Documentación técnica** completa en cada módulo

### Qué he aprendido
Integrar tecnologías de distintos ámbitos (red, cloud, base de datos, web) en un proyecto unificado. Tomar decisiones técnicas sin supervisión directa. Aplicar conocimientos que aún no están en el temario cuando el proyecto los requiere.
